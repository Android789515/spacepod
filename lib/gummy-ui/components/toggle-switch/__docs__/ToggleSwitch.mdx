# ToggleSwitch

ToggleSwitch component with different props.

#### Example

<Canvas of={ToggleSwitch.Off} />

## Usage

```ts
import { ToggleSwitch } from 'gummy-ui';

export const Component = () => {
  return (
      <ToggleSwitch
        fontSize='1.2rem'
        text='toggle me'
        toggled={false}
        width='8em'
        knobColors={{
          off: 'light-dark(#b8b8b9ff, #4d4c4d)',
          on: '#e30fb2'
        }}
        background='light-dark(#ebeaf3, #4d4c4d)'
        borderRadius='.25em'
      />
  );
};
```

#### Arguments

- **fontSize** _`string`_ Font size of the text.
- **text?** _`string`_ - An optional string that represents the text content of the toggle switch.
- **toggled** - A boolean indicating whether the switch is toggled.
- **width** _`string`_ - Width for the toggle switch.
- **knobColors** _`{ off: string, on: string }`_ - Colors for the knob in both on and off states.
- **background?** _`string`_ - An optional background for the toggle switch.
- **borderRadius?** _`string`_ - An optional border radius (defaults to var(--borderRadius)).
